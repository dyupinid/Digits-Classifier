# Digits-Classifier
Классификатор MNIST на PyTorch

Обучение нейронной сети распознаванию рукописных цифр из датасета MNIST. Включает загрузку данных, визуализацию, определение модели, обучение, оценку точности, сохранение модели и демонстрацию предсказания.
Использование torchvision для датасета MNIST с нормализацией. Модель — многослойный персептрон (MLP) с двумя скрытыми слоями (512 нейронов каждый). Обучение идёт 5 эпох с Adam-оптимизатором и CrossEntropyLoss.Есть расчёт точности на тестовом датасете (около 97%)
Модель сохраняется в файл mnist_model.pth
